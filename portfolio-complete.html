A Challenge Encountered During NPI

Challenge

BIOS and BMC EEPROM programming flow during NPI: aligning board-level and system-level manufacturing sequences

Background

Some team members had come from the laptop industry, where firmware configurations are typically finalised early in the design stage. In those products, the flash is soldered directly onto the PCB, programmed on the board side, and rarely changed afterward.

Server platforms operate differently, and this gap in mental model created a discussion the team needed to resolve before production flows could be finalised.

The technical context

BIOS EEPROM

Reads configuration values from EEPROM based on sample code: board ID, PCIe card configuration, and other system settings.

BMC EEPROM

Controls system management parameters: fan table, temperature monitoring thresholds, and fan speed control. These may need adjustment after thermal testing results come in.

Two options on the table

A

Program EEPROM at board level

Program the BMC EEPROM during board-side functional testing, before system assembly.

Pro Simpler manufacturing flow for the board team
Con May require reprogramming if thermal parameters change after thermal validation
B

Program EEPROM at system level

Program the finalised BMC EEPROM parameters during system-level production testing, after full assembly.

Pro Final thermal configuration applied only after validation is complete
Con Slightly more complex system-level test flow

My resolution

A+B

A hybrid approach: board-level testing uses default values for basic hardware verification, while system-level production testing applies the finalised BMC EEPROM parameters validated through thermal testing.

This allowed the board team to complete functional verification earlier in the schedule, without waiting on thermal test results. The system team could then apply the correct fan table and temperature thresholds as a final step during production. It preserved schedule efficiency at both stages while ensuring the shipped configuration was thermally validated.
